About This Portfolio

A modern, responsive portfolio website for an AI Engineer/Robotics Enthusiast. Features a stunning 3D robot animation, dark/light mode toggle, and a project slider/carousel.

FEATURES:

- Dark/Light Mode Toggle - User preference saved in browser
- 3D Animated Robot - Interactive Three.js model in background
- Fully Responsive- Works on desktop, tablet, and mobile
- Project Slider/Carousel- Horizontal scroll with navigation buttons
- Contact Form - Frontend demo with console logging
- Social Links - GitHub, LinkedIn, Email
- Smooth Scrolling - Animated navigation between sections

TECHNOLOGIES USED

- HTML5 - Semantic structure
- CSS3 - CSS Variables, Flexbox, Grid, Animations
- JavaScript - Dark mode, slider, smooth scroll
- Three.js - 3D robot model and particle effects
- Font Awesome 6 - Icons
- Google Fonts - Inter & Space Grotesk
